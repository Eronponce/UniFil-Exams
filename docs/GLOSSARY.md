# Glossary

## Disciplina
Area ou materia que organiza um banco de questoes.

## Questao Objetiva
Questao com enunciado, cinco alternativas A-E e uma alternativa correta.

## Auditoria
Revisao humana da questao antes de uso em prova.

## Set
Versao da prova, como A, B ou C, com ordem propria de questoes e alternativas.

## Randomizacao
Embaralhamento da ordem das questoes e/ou alternativas.

## Gabarito
Lista de respostas corretas de um set depois da randomizacao.

## EvalBee
Software externo usado pelo usuario para correcao por folha/resposta escaneada. Na V1, o app apenas anexa a imagem EvalBee preparada pelo usuario ao PDF.

## Qwen Local
Modelo local Qwen3.5 9B Q4_K_M usado para gerar questoes sem depender de API externa.

## Provedor IA
Origem usada para gerar questoes: Qwen local, Claude API ou Gemini API.
